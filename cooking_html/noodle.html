레시피 출처: 1분요리 뚝딱이형

< 양념장 재료 >

4인분 기준

(계량 : 밥 숟가락, 종이컵)

조리도구 : 믹서기 필요

사과 : 1/2개

양파 : 1/4개

칠성 사이다 : 1컵 (종이컵 기준, 180ml)

고추장 : 2숟가락

진간장 : 2숟가락

고춧가루 : 2숟가락

설탕 : 2숟가락

물엿 : 2숟가락

식초 : 2숟가락

청양고추 : 1개


< 그 외 토핑재료 >

1인분 기준

소면 : 1인분

시판 초고추장 : 1숟가락

다진 김치 : 1~2숟가락

상추 : 조금

김가루 : 조금

깨 : 1숟가락

참기름 (또는 들기름) : 1숟가락



[조리 과정]

1. 믹서기에 아래의 재료를 모두 넣고 곱게 갈아준다.

1) 사과 1/2개

2) 양파 1/4개

3) 칠성 사이다 1컵 (종이컵 기준, 180ml)

4) 고추장 2숟가락

5) 진간장 2숟가락

6) 고춧가루 2숟가락

7) 설탕 2숟가락

8) 물엿 2숟가락

9) 식초 2숟가락

10) 청양고추 1개

2. 소면 또는 중면을 먹을만큼 삶아 찬물에 행군 뒤 두 손가락으로 국수를 돌돌말아 예쁘게 세팅한다.

>

3. 시판 초고추장 1숟가락을 국수 위에 올리고 만들어둔 비빔국수 양념장 8숟가락을 올려준다.

4. 다진 김치 1~2숟가락을 올린 뒤 기호에 따라 상추 등의 채소와 김가루를 올려준다.

5. 통깨 1숟가락을 절구에 넣고 빻아서 깻가루를 만들어준다. 깨를 절구에 갈아서 넣으면 그 고소함이 극대화된다.

5. 통깨 1숟가락을 절구에 넣고 빻아서 깻가루를 만들어준다. 깨를 절구에 갈아서 넣으면 그 고소함이 극대화된다.

완성된 비빔국수!
